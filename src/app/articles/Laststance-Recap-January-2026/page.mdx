## corelive

A real-time todo/task management application with web and Electron desktop support.

**Highlights:**

- [Implement drag-and-drop reordering for todos](https://github.com/laststance/corelive/commit/7ec9193327595f1cc26b1503866d78e7ebe98f07)
  - Added `@dnd-kit` integration for smooth drag-and-drop task reordering with optimistic updates and server synchronization. This significantly improves the task management UX.

- [Migrate main process to TypeScript and add Settings with Hide App Icon](https://github.com/laststance/corelive/pull/14)
  - The Electron main process is now fully TypeScript, and users can hide the app icon from the dock for a cleaner desktop experience.

- [Add eslint-plugin-dslint for design token enforcement](https://github.com/laststance/corelive/commit/a24bb51ee104e14c80c398e39cb1aaab06efa345)
  - Created a custom ESLint plugin that enforces design system compliance using Tailwind config as the single source of truth. This ensures consistent styling across the entire application.

**Releases:** v0.1.0 through v0.1.8 (9 releases)

---

## gitbox

A GitHub repository management dashboard with Kanban-style organization.

**Highlights:**

- [Migrate from MSW to real local Supabase database for E2E tests](https://github.com/laststance/gitbox/pull/45)
  - Replaced MSW mock handlers with a real local Supabase instance for higher test confidence. This ensures E2E tests reflect actual database behavior.

- [Migrate project structure to src directory](https://github.com/laststance/gitbox/pull/40)
  - Reorganized the entire project following Next.js conventions, moving all source files into a `src/` directory for better organization and maintainability.

- [Migrate forwardRef to React 19 ref-as-prop pattern](https://github.com/laststance/gitbox/commit/913ab2644ecfd8f18982f043bc6d489e9fcdcc12)
  - Updated all UI components to use React 19's new ref-as-prop pattern, removing the need for forwardRef wrappers and modernizing the codebase.

**Releases:** v0.1.0, v0.2.0

---

## claude-plugin-dashboard

A TUI (Terminal User Interface) dashboard for managing Claude Code plugins and marketplaces.

**Highlights:**

- [Add component detail view with focus mode navigation](https://github.com/laststance/claude-plugin-dashboard/pull/20)
  - Users can now drill into plugin components (Skills, Slash commands, Agents, MCP servers) and view detailed information with Emacs-style keyboard navigation.

- [Add marketplace management capabilities](https://github.com/laststance/claude-plugin-dashboard/pull/15)
  - Full marketplace management with add (`a`), remove (`d`), and update (`u`) keyboard shortcuts. Supports GitHub shorthand, Git URLs, and local paths.

- [Display plugin component names in detail panel](https://github.com/laststance/claude-plugin-dashboard/pull/18)
  - Plugin details now show actual component names instead of just counts, making it easier to understand what each plugin provides.

**Releases:** v0.1.1, v0.2.0, v0.2.1, v0.2.2, v0.2.3, v0.3.0 (6 releases)
